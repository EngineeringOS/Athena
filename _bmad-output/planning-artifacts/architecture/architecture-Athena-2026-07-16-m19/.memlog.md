- (event) started M19 architecture spine drafting
- (decision) Adopt semantic projection pipeline as the M19 paradigm: governed semantic authority feeds deterministic sheet projection and layout facts before any rendering surface consumes them.
- (constraint) M19 stays a single shared milestone-level spine aligned to the entire M19 story set; it is not a separate architecture per individual story or view family.
- (decision) Make the schematic sheet the primary customer-facing workflow: frame, grid, title block, labels, terminals, conductors, and cross references all come from projection rather than frontend reconstruction.
- (decision) Keep selection, reveal, inspector, and diagnostics synchronized through canonical subject identity so the IDE surfaces reflect the same governed subject.
- (decision) Treat layout facts as deterministic projection outputs; the renderer may place and paint them but may not invent meaning or re-resolve semantics.
- (decision) Allow a cabinet preview only as a secondary read-only view if it can reuse the same contracts cleanly; otherwise defer it.
- (decision) Defer full EPLAN parity, IEC library breadth, public repository or import ecosystem work, and frontend-owned semantic resolution out of M19.
- (decision) Keep the proof corpus small, executable, and governed by local fixtures so M19 can demonstrate the customer-facing workflow without a large refactor surface.
- (event) spine finalized
- (decision) Introduce a first-class sheet IR between projection and rendering so page size, frame, coordinate zones, title block, revision metadata, and view composition are owned by Athena instead of the canvas.
- (decision) Frame M19 as the first professional engineering sheet workflow rather than a generic visibility milestone so product language tracks publication credibility.
- (decision) Assume the first proof corpus uses page-anchored, grid-assisted placement with mostly orthogonal conductor routing to keep sheet output deterministic and reviewable.
- (change) Aligned AD-5 and capability map with the PRD decision that cabinet preview is deferred from M19.
