- (event) Architecture create run started from the Athena M2 PRD on the fast path with the spine itself treated as the main builder deliverable.
- (constraint) Inherited repo reality remains binding from M0 and M1: Athena Runtime owns lifecycle and orchestration, Engineering IR remains canonical semantic authority, commands remain the only semantic mutation path, Compose stays downstream, and plugins remain typed and non-sovereign.
- (decision) M2 paradigm chosen: single-process layered semantic runtime with compiler-derived projection pipeline and geometry-backed multi-view surfaces, preventing layout and geometry from being re-created ad hoc in UI or renderer code.
- (decision) M2 introduces two explicit downstream kernel contracts, Layout IR and Geometry IR, both derived from Engineering IR while preserving canonical semantic identity across projection layers.
- (decision) Compiler-owned deterministic projection derivation produces Layout IR then Geometry IR; Athena Runtime owns active projection sessions, view switching, refresh coordination, and projection inspection over active projects.
- (decision) Supported views are runtime-activated but extension-contributed contracts; the first M2 proof pair is cabinet view plus wiring view contributed by the electrical extension, preventing view logic from being hard-coded inside the desktop UI.
- (decision) Geometry IR becomes the only renderer-facing contract in M2: SVG and Compose surfaces consume geometry-backed projection snapshots and may not privately reconstruct layout or geometry from Engineering IR.
- (decision) Projection refresh remains dependency-scoped and runtime-explained: semantic commands identify affected canonical identities first, then only impacted view/layout/geometry scope is recomputed.
- (decision) M2 seed module growth adds kernel/layout-model and kernel/geometry-model for durable projection contracts while keeping orchestration in kernel/runtime and deterministic derivation in kernel/compiler for the first proof.
- (version) Brownfield stack ratified from the repository state on 2026-07-06: Java 25, Kotlin 2.4.0, Gradle 9.6.1, Compose Multiplatform 1.11.1, with versions managed through gradle/libs.versions.toml and the Gradle wrapper.
- (direction) Web/WASM delivery, arbitrary manual layout authoring, geometry serialization as a published interchange artifact, and external layout/target adapters remain deferred beyond the first M2 proof.
