# Glossary

- **Athena** - The implementation platform that operationalizes the EngineeringOS thesis.
- **Engineering Language** - The human-readable source language for expressing engineering intent.
- **Engineering IR** - The first and only canonical semantic model in M0.
- **Engineering Ontology** - The governed conceptual vocabulary behind semantic meaning.
- **Engineering Compiler** - The pass-based execution substrate that validates, derives, and prepares outputs from `Engineering IR`.
- **Electrical/Runtime domain extension** - The first domain-specific layer built on top of the general semantic core.
- **Render-facing model** - The downstream view model derived from `Engineering IR` for rendering targets such as `SVG`.
- **Plugin** - A typed extension discovered locally by manifest and classpath, attaching only through core-owned contracts.
- **Conformance artifact** - An example with stable expected validation, `Engineering IR`, and output behavior that defines the proof contract.
