# M10 Epic 2 Retrospective

## What Worked

- One deterministic proof provider was enough to prove grounded explanation, impact, and next-check flows.
- Reusing M9 evidence and M6 review state avoided inventing a second semantic path.

## What Needed Tightening

- Grounding is only as good as evidence selection, so proof-provider output had to cite ids directly.

## Carry Forward

- Live providers must keep the same cited-evidence discipline.
- New categories should reuse the same audit model instead of new side channels.
