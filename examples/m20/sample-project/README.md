# M20 Sample Project

This is the openable Athena workspace for the M20 milestone.

Open this folder in the IDE to see the finished M20 scenarios as real `.athena` source files:

- `src/01-schematic-sheet.athena` - governed schematic sheet baseline
- `src/02-dense-sheet.athena` - dense layout and readability case
- `src/03-acceptance-sheet.athena` - customer-facing acceptance slice
- `src/04-boundary-scope.athena` - lean boundary slice using the same governed contracts

The `.mjs` fixtures under `examples/m20/` remain the executable proof corpus. This workspace is the
customer-facing sample project that the IDE can open directly.
