# Guided Authoring Proof

This repository is the narrow M15 proof corpus for guided semantic authoring.

Use it to verify the first governed flow:

1. Open the repository in Athena.
2. Insert one PLC CPU from the component panel.
3. Rename the inserted PLC through the inspector.
4. Insert one power supply from the component panel.
5. Use graph connect mode to connect compatible power ports.
6. Accept each guided preview and verify source, graph, inspector, and semantic SCM stay aligned.
