# M23 Examples

M23 proves governed layout hint language admission.

- `sample-project/` is the openable IDE workspace for M23.
- `sample-project/src/01-layout-hints.athena` contains the real system-scoped layout block admitted by ANTLR, Tree-sitter, compiler, and LSP.

The `.mjs` files in the repository are tests only. Review M23 from the Athena IDE by opening the sample project.
