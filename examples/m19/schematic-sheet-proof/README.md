# M19 Schematic Sheet Proof

This fixture is a small governed diagram payload used by the Theia frontend proof tests.

It represents one schematic sheet with:

- a stable active sheet id
- title block inputs from governed sheet metadata
- one component, two terminal labels, and one orthogonal conductor route
- a repeated-reference marker tied to canonical subject identity

The fixture is intentionally small. It proves deterministic sheet publication and visible schematic structure without expanding M19 into cabinet preview, public repository/import ecosystem work, broad IEC library ingestion, or final protocol/layout-stack selection.
