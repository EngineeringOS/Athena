# M19 Proof Corpus

This directory contains the local executable proof corpus for the M19 professional schematic sheet milestone.

- `schematic-sheet-proof/` proves a small governed schematic sheet fixture with page chrome, title block, terminals, conductor routing, and cross-reference markers.

The corpus is local and schematic-first. It does not imply cabinet preview, remote registry, marketplace, publish, IEC library ingestion, frontend-owned semantic resolution, desktop-viewer, or protocol/layout-stack selection behavior.
